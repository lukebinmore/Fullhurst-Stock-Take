## Overview
This is a repository housing all code found in the Fullhurst Community College PACA Inventory Tracker, as well as a collection of guides and instructions for it's use. These macro's and scripts are written Visual Basic, and are used inside of Excel to allow simpler but more in-depth tracking of product and equipment stock.
